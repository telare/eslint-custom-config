# @tl21/eslint-config

My personal concise ESLint configuration for modern **React/Next.js + TypeScript** projects.
Designed for **Flat Config** (ESLint 9+).

## Features

* **React**: Full JSX/TSX support. 
* **TypeScript**: Recommended `typescript-eslint` rules.
* **Best Practices**: Standard JS & TS recommended rules.

## Requirements
* eslint 9.0.0+

## Installation

```bash
npm install -D eslint @tl21/eslint-config
```
